std::vector< int > &  v 
)

This method maps the leaf nodes from left to right in the input index set. Mapping set is used to permute the original matrix so that clustered rows and cols are together, which is important when constructing the block cluster tree. It also modifies the index tree according to the index set to create an ascending tree at each level.

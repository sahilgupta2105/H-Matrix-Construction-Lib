Hierarchical Matrix construction library

Introduction

The library is used to convert an input matrix to the corresponding Hierarchical Matrix. The library uses an algebraic approach based on graph partitioning algorithms to construct a cluster tree. This approach requires no information about the underlying domain of the problem. The library should only be used for a square-symmetric matrix. 'main.cpp' is the front-end interface for the user.

Supporting modules

  1. graph_cluster.cpp: This handles the graph clustering process which is later used to build cluster tree.
  2. tree.cpp: This is used to handle index tree and cluster tree; other supporting modules inside main.cpp are used to construct the tree.
  3. block_cluster.cpp: This handles the block cluster tree building process.
  4. h_mat.cpp: This handles Hierarchical Matrix building process and contains functions for compressing matrix blocks as Rk matrix.

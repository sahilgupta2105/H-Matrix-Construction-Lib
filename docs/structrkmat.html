Detailed Description

This class can be used for a block cluster tree.

Three structs for handling the blocks during the partition process. The structs are described below: rkmat: used for handling R-K Matrix blocks. fullmat: used for handling leaves of the block cluster tree which are stored as full matrices. supermat: used for handling the blocks in process of partitioning. rkmat: k: expected rank of rk block (same as input rank 'r'). kt: rank of the rk block a: stores the column vectors. b: stores the row vectors. Note: both 'a' and 'b' are stored as row vectors (due to some error in Eigen library). Care to be taken when performing operations on rk blocks.
